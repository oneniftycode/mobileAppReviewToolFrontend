About Dynamic Analysis

Dynamic analysis is one the many testing and evaluations security professionals utilize in their day to day work. The main objective of this type of analysis is to find errors in a program while it is running, rather than by doing manual code review. Dynamic Analsysis tests a program by executing it's functions and services in real-time. The objective is to find possible malfunctioning services and expose security holes.
